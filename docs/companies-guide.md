# Şirketler İçin Rehber

Bu rehber, staj fırsatlarınızı paylaşmak isteyen şirketler için hazırlanmıştır.

## 📋 Nasıl Başvuru Yapılır?

### 1. Issue Oluşturma

1. Repository'nin ana sayfasına gidin
2. **Issues** sekmesine tıklayın
3. **New Issue** butonuna basın
4. **"Şirket Staj Fırsatı"** template'ini seçin

### 2. Formu Doldurma

Formu doldururken dikkat edilmesi gerekenler:

- **Şirket Adı**: Tam ve resmi şirket adınızı yazın
- **Konum**: Şirketinizin bulunduğu şehir/şehirler
- **Sektör**: Faaliyet gösterdiğiniz sektör (Yazılım, Finans, vb.)
- **Website**: Şirket web sitenizin linki (varsa)
- **Staj Dönemi**: Yaz, Kış veya Tüm Yıl
- **Staj Süresi**: Minimum ve maksimum süre bilgisi
- **Başvuru Dönemi**: Hangi aylarda başvuru kabul ediyorsunuz
- **Başvuru Linki**: Kariyer sayfanız veya başvuru formu linki
- **Staj Ücreti**: Ücretli/ücretsiz bilgisi

### 3. Pozisyon Seçimi

En az bir pozisyon seçmeniz gerekmektedir. Seçenekler:

- Mobile
- Backend
- Frontend
- PM (Product Manager)
- QA
- Game Development
- Data Science
- Data Analyst
- Database
- Embedded
- Cyber Security
- Blockchain
- System
- Networking
- Hardware
- SAP ABAP

### 4. Staj Tipi

- **Uzaktan**: Tamamen uzaktan çalışma
- **Yüzyüze**: Ofiste çalışma
- **Hibrit**: Hem uzaktan hem yüzyüze

### 5. Gereksinimler

Stajyer adaylarınızdan beklediğiniz gereksinimleri detaylı olarak belirtin:

- Teknik gereksinimler (programlama dilleri, frameworkler)
- Eğitim seviyesi
- Deneyim gereksinimleri
- Diğer özellikler

### 6. Ek Bilgiler

Şirketinizin sunduğu ek imkanları belirtin:

- Yemek imkanı
- Ulaşım desteği
- Eğitim ve gelişim fırsatları
- Diğer avantajlar

## ✅ Başvuru Sonrası Süreç

1. **Otomatik Validasyon**: Başvurunuz otomatik olarak kontrol edilir
2. **Onay**: Eksiklik yoksa başvurunuz onaylanır
3. **Listeye Ekleme**: `usefull_company.md` dosyasına eklenirsiniz
4. **Eşleştirme**: İlgili adaylara otomatik bildirim gönderilir

## 🎯 Eşleştirme Sistemi

Sistem otomatik olarak:

- Şirketinizin ihtiyaçlarına uygun adayları bulur
- Size en uygun adayları önerir
- Adaylara şirketiniz hakkında bildirim gönderir

## 📝 Örnek Başvuru

```markdown
### Şirket Bilgileri

- **Şirket Adı**: Örnek Yazılım A.Ş.
- **Konum**: İstanbul
- **Sektör**: Yazılım
- **Website**: https://www.ornek.com

### Staj Detayları

- **Staj Dönemi**: Yaz
- **Staj Süresi**: 4-6 hafta
- **Başvuru Dönemi**: Mart-Nisan
- **Başvuru Linki**: https://www.ornek.com/kariyer
- **Staj Ücreti**: Ücretli

### Pozisyonlar

- [x] Backend
- [x] Frontend
- [ ] Mobile

### Staj Tipi

- [x] Uzaktan
- [x] Yüzyüze
- [x] Hibrit

### Gereksinimler

- Python veya JavaScript bilgisi
- Temel veritabanı bilgisi
- Takım çalışmasına yatkınlık

### Ek Bilgiler

- Öğle yemeği ücretsiz
- Ulaşım desteği mevcut
- Mentorluk programı
```

## ❓ Sık Sorulan Sorular

### Başvurum ne zaman onaylanır?

Başvurunuz eksiksizse otomatik olarak onaylanır ve listeye eklenirsiniz.

### Kaç pozisyon belirtebilirim?

İstediğiniz kadar pozisyon belirtebilirsiniz. En az bir pozisyon seçmeniz gerekmektedir.

### Başvurumu güncelleyebilir miyim?

Evet, issue'yu düzenleyerek bilgilerinizi güncelleyebilirsiniz.

### Adaylara nasıl ulaşabilirim?

Sistem otomatik olarak size uygun adayları bulur ve size önerir. Ayrıca adayların issue'larına yorum yaparak iletişime geçebilirsiniz.

## 📞 İletişim

Sorularınız için issue açabilir veya repository maintainer'ına ulaşabilirsiniz.
